# Data

This project currently runs on a **synthetic dataset** generated inside the
notebooks — there are no data files to download to reproduce the results.

This folder is a placeholder for **real data** if you extend the project later.
Raw data files (`.csv`, `.xlsx`, `.nc`) are git-ignored to keep the repo light;
document their sources here instead.

## Suggested sources (free / open)

| Data | Source |
|------|--------|
| Demand & generation actuals | ENTSO-E Transparency Platform (free API token) |
| Solar resource (hourly PV) | PVGIS (EU) or renewables.ninja |
| Plant fleet & capacities | TSOC annual reports / Global Energy Monitor |
| Fuel & CO2 prices | EU ETS price + published fuel cost assumptions |
| Geographic coordinates | OpenStreetMap / manual |
